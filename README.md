# AES-256
Implementation of Advanced Encryption Standard. 

MODE : CBC 

Key Size : 256bits 


For more Implementation Details, read PDF : 
https://engineering.purdue.edu/kak/compsec/NewLectures/Lecture8.

# USAGE

git clone https://github/addy1397/AES-256/

cd AES-256

mkdir build

cmake ..

make

./AES-256
